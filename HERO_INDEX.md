**A:** Look at the Live Preview at the top of the form - it updates in real-time!

### **Q: Can I reorder slides?**

### **Q: How do I delete a slide?**

## üìû **Support & Troubleshooting**

### **Issue: Preview not updating**
**Solution:** Check browser console for errors, refresh page

### **Issue: Colors look wrong**
**Solution:** Ensure hex codes start with # (e.g., #ffffff)

### **Issue: Image not uploading**
**Solution:** Check file size (<5MB) and format (JPG/PNG)

### **Issue: Changes not saving**
**Solution:** Check Firebase connection, look for error toast

### **Issue: Homepage not updating**
**Solution:** Refresh homepage, verify slide is enabled
